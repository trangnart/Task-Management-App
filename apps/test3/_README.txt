Index -
    The first user signing up with be added role as 'ceo'

    all other users after will have to select a manager before they can view the index page

    managers must exist as registered users of task_mgmt before their employees can select them as a manager 


Select and change own manager -
    Selecting a manager is requried for all users except for the first one signing up. All registered 
    users (except for first one) will be prompted to select a manager before they can view/create tasks

    changing your manager is done under the user menu on the top right under "Edit Manager"
